# AuroraWall

AuroraWall is an open source video wallpaper project for Ubuntu + GNOME, focused on security, minimal permissions, and community-driven development.

## Goals
- Ubuntu + GNOME first
- Security-first architecture
- Minimal permissions
- Modular design for future desktop environments
- Open source from day one

## Project structure
- `app/`: desktop application
- `gnome-extension/`: GNOME integration layer
- `docs/`: architecture, security, roadmap
- `assets/`: images and branding
